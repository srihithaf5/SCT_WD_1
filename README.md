# SkillCraft Technology - Task 01

## Responsive Landing Page

This project is developed as part of the SkillCraft Technology Web Development Internship.

### Objective

Create an interactive navigation menu that:

- Has a fixed position and remains visible on all pages.
- Changes color or style when scrolling.
- Changes appearance when hovering over menu items.
- Is fully responsive across different screen sizes.

---

## Features

✅ Fixed Navigation Bar

✅ Navigation Bar Changes Color on Scroll

✅ Hover Effects on Menu Items

✅ Responsive Design

✅ Hero Section

✅ About Section

✅ Services Section

✅ Portfolio Section

✅ Contact Form

✅ Modern UI Design
